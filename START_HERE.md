## ⚡ Quick Commands Reference

```bash
# Start server
php artisan serve

# Stop server
Ctrl+C

# Run migrations
php artisan migrate

# Seed data
php artisan db:seed

# Reset database
php artisan migrate:fresh --seed

# Clear cache
php artisan cache:clear

# Build assets
npm run build

# Watch mode (dev)
npm run dev
```

## 🆘 If Something Goes Wrong

### "Database error"
```bash
php artisan migrate
php artisan db:seed
```

### "Assets not loading"
```bash
npm run build
php artisan cache:clear
```

### "Port 8000 in use"
```bash
php artisan serve --port=8001
```

## 🎓 Learning Notes

- The code is well-commented and easy to understand
- Follow the MVC pattern (Model-View-Controller)
- Blade templates are in `resources/views/`
- Database logic is in `app/Models/`
- Request handling in `app/Http/Controllers/`
- Routes defined in `routes/web.php`
